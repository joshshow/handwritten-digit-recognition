# handwritten-digit-recognition
Trained a Convolutional Neural Network to identify handwritten digits

Technologies Used
- Python
- Tensorflow
- Keras
- Matplotlib
- Numpy

How it works
- Training samples were loaded from Keras
- Those samples were normalized (Max Normilization) to produce more accurate results
- Passed samples through a Convolutional Neural Network (CNN) to instruct it on digit classification
